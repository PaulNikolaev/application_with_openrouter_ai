# Сборка десктопного приложения

Инструкция по сборке исполняемого файла для Windows и Linux.

## Системные требования

- Windows 10/11 или Linux
- Python 3.7 или выше
- pip (менеджер пакетов Python)
- Минимум 2 ГБ свободного места на диске

## Установка зависимостей

Убедитесь, что виртуальное окружение активировано и зависимости установлены:

```
pip install -r requirements.txt
```

## Сборка для Windows

1. Перейдите в директорию проекта
2. Запустите сборку:
```
python build.py
```
3. Исполняемый файл: `AIChat/AIChat.exe`

## Сборка для Linux

1. Перейдите в директорию проекта
2. Запустите сборку:
```
python3 build.py
```
3. Исполняемый файл: `AIChat/aichat`
4. Сделайте файл исполняемым:
```
chmod +x AIChat/aichat
```

## Результат сборки

Все файлы приложения создаются в директории `AIChat/` в корне проекта.

## Решение проблем

- Убедитесь, что установлен PyInstaller: `pip install pyinstaller`
- Проверьте наличие свободного места на диске
- При сборке PyInstaller создает временные папки `build/` и `dist/` - их можно удалить после сборки
- Проверьте вывод консоли на наличие ошибок
